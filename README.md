# 基于vue-cli 3的定制脚手架命令行工具

## 用法

#### 1、安装vue-cli
`npm install -g @vue/cli`  

#### 2、安装my-vue-cli
`npm install -g vuecli-preset`  

### 3、在要创建项目的目录下
`my-vuecli go 你的项目名`
